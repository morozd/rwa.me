Alarmik is a concept I've been exploring since January 2013. I haven't set a deadline for myself, and don't want to rush things. One of the reasons I've been holding off development for a while is that as of right now it's still not possible to set a real alarm in iOS7. You can only play a sound that ignores the 'silent switch' of the iPhone if your app is running in the background. I hope Apple will give us access to setting an actual alarm in the future so third party alarm clock apps don't need to give the user a big set of instructions to set an alarm that will actually wake them in the morning..
